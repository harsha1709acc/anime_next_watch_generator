# Anime Next Watch Generator

A sleek, modern web application designed for finding your next anime obsession. Features a seamless dark-theme and glassmorphism interface.

## Features
- **Trending Now**: Pulls real-time trending anime from Jikan API.
- **Top Rated**: Dynamically loads the highest-rated anime series.
- **Genre Search**: Filter through top action, romance, comedy, and more.
- **Micro-Animations**: Clean hover and expansion interactions for anime synopsis cards.

## Built With
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla DOM interaction + Fetch API)
- [Jikan API v4](https://docs.api.jikan.moe)
